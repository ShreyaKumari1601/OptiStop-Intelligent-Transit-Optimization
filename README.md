# OptiStop: Intelligent Transit Stop Optimization System (Advanced Version)

## 📌 Problem Statement
Urban public transport systems often suffer from inefficient bus stop placements,
leading to longer commuter walking distances and uneven demand coverage.

## 🎯 Objective
To use Machine Learning-based geospatial clustering techniques to determine optimal
bus stop locations based on commuter demand density.

## 🧠 Methodology
- Synthetic commuter demand generation (Bangalore region simulation)
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Elbow Method for optimal cluster selection
- Silhouette Score evaluation
- Interactive geospatial visualization using Folium

## 📊 Features
- Demand density analysis
- Optimal stop identification
- Cluster evaluation metrics
- Interactive HTML decision-support map

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python main.py

3. Open the generated file:
   outputs/optimized_stops_map.html
